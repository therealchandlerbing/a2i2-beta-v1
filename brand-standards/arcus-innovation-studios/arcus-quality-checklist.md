# Arcus Innovation Studios - Quality Checklist

**Pre-delivery validation for all Arcus-branded materials**

---

## How to Use This Checklist

Before delivering ANY Arcus-branded material, verify ALL items in the relevant sections. Every checkbox must be checked (✓) before delivery.

**Failure to meet any item = DO NOT DELIVER until fixed.**

---

## Universal Requirements (All Documents)

**Must pass ALL items regardless of document type:**

### Brand Color Compliance

- [ ] **All colors from Arcus approved palette** (no generic blues, grays, or other non-brand colors)
- [ ] **Navy (#0A2540) used for all primary headers** (not black, not other blues)
- [ ] **Teal (#00BFB3) used strategically for accents** (not overused, not primary)
- [ ] **Charcoal (#1A1F36) used for body text** (not pure black)
- [ ] **Gray scale from Arcus palette** (Gray 50, 100, 200, 300, 600, 700, 900)
- [ ] **Maximum 2-3 colors per section** (not overwhelming)
- [ ] **Accent colors used appropriately** (Green for success, Amber for warnings, Red for errors)

### Typography Compliance

- [ ] **System font stack used** (Apple system fonts, not custom fonts)
- [ ] **Document title 32-36px, Navy, 700 weight** (clear hierarchy)
- [ ] **Section headers 24-28px, Navy, 600-700 weight** (proper scale)
- [ ] **Subsection headers 18-20px, Slate or Navy, 600 weight** (clear distinction)
- [ ] **Body text 14px minimum, Charcoal, 400 weight** (readable)
- [ ] **Captions 12px, Gray 600, 400 weight** (supporting info)
- [ ] **Line height 1.7 for body text** (generous spacing for readability)
- [ ] **Line height 1.2-1.4 for headers** (tighter for impact)
- [ ] **Letter spacing -0.02em for large text** (titles, headers)
- [ ] **Hierarchy is 2.5-3x between title and body** (clear visual separation)

### Logo Compliance

- [ ] **Logo included** (if external document - proposals, contracts, presentations, letterhead)
- [ ] **Logo placement correct for document type** (see placement matrix)
- [ ] **Logo size meets minimum** (0.75" print / 120px digital)
- [ ] **Clear space maintained** (height of "A" on all sides, no violations)
- [ ] **No logo distortion** (proportions unchanged)
- [ ] **No logo effects** (no shadows, glows, gradients, outlines, or filters)
- [ ] **Correct logo variation used** (Navy on White OR White on Navy, not Teal unless digital-only)
- [ ] **Logo not placed on busy background** (white or navy container if needed)

### Spacing Compliance

- [ ] **8-point grid followed throughout** (4, 8, 16, 24, 32, 48, 64, 96px)
- [ ] **Paragraph spacing 16px minimum** (readable content)
- [ ] **Section margins 48-64px** (clear divisions)
- [ ] **List item spacing 8px** (vertical rhythm)
- [ ] **Card/callout padding 24-32px** (breathing room)
- [ ] **Page margins appropriate** (1" standard, 1.25" top for letterhead)
- [ ] **Generous white space for C-level audience** (not cramped)
- [ ] **Consistent spacing relationships** (same spacing for same element types)

### Accessibility Compliance (WCAG AA)

- [ ] **Color contrast 4.5:1 minimum for body text** (Navy on white = 16.1:1 ✓)
- [ ] **Color contrast 3:1 minimum for large text (18pt+)** (Teal on white = 3.6:1 ✓)
- [ ] **Text size 14px minimum for body** (readable)
- [ ] **Focus states present on interactive elements** (web only, 2-3px outline)
- [ ] **Touch targets 44x44px minimum** (web only)
- [ ] **Color not sole indicator of meaning** (icons or text labels included)
- [ ] **Alt text on all images** (descriptive, contextual)
- [ ] **Semantic structure** (proper heading hierarchy, lists, etc.)

### Professional Quality

- [ ] **Hand-crafted appearance** (not template-based or generic)
- [ ] **Board-ready quality** (would pass C-level scrutiny)
- [ ] **Professional tone throughout** (appropriate for executive audience)
- [ ] **No spelling or grammar errors** (proofread completely)
- [ ] **No placeholder text** ([CONTENT HERE] or similar)
- [ ] **Consistent design application** (all pages/slides match)
- [ ] **Appropriate file format** (PDF for final, DOCX for editable)

## Common Issues & Fixes

### Issue: Checklist item fails

**Action:** DO NOT DELIVER. Fix the issue first.

### Issue: Multiple checklist items fail

**Action:** Re-apply Design Director Phase 4 (Systematic Application) with Arcus brand elements. Then re-validate.

### Issue: Colors are close but not exact Arcus hex codes

**Action:** Replace ALL colors with exact Arcus palette hex codes:
- Navy: `#0A2540`
- Teal: `#00BFB3`
- Slate: `#425466`
- Charcoal: `#1A1F36`
- Gray scale: `#FAFBFC`, `#F3F4F6`, `#E5E7EB`, `#D1D5DB`, `#6B7280`, `#425466`, `#1A1F36`

### Issue: Typography sizes don't match Arcus standards

**Action:** Systematically update ALL text:
- Titles: 32-36px Navy 700
- Headers: 24-28px Navy 600-700
- Subsections: 18-20px Slate 600
- Body: 14px Charcoal 400, 1.7 line height
- Captions: 12px Gray 600 400

### Issue: Logo missing, wrong size, or placed incorrectly

**Action:**
1. Determine correct placement from document type matrix
2. Add logo at correct size (minimum 0.75" print / 120px digital)
3. Ensure clear space (height of "A" on all sides)
4. Use correct variation (Navy on White or White on Navy)
5. No effects or distortion

### Issue: Spacing feels inconsistent or cramped

**Action:**
1. Apply 8-point grid strictly (4, 8, 16, 24, 32, 48, 64, 96px)
2. Increase section margins to 48-64px
3. Ensure 16px paragraph gaps
4. Add breathing room around key elements
5. Review against Arcus spacing standards

### Issue: Doesn't feel "board-ready"

**Action:**
1. Increase formality (Navy + Teal + White only)
2. Add more white space
3. Ensure conservative, professional tone
4. Remove any casual or decorative elements
5. Verify executive summary is prominent (if applicable)
6. Check all professional quality items
