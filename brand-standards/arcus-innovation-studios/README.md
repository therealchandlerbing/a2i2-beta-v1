# Arcus Innovation Studios Brand Module

**Complete brand standards integration for Design Director skill**

---

## Overview

This brand module provides comprehensive brand standards for Arcus Innovation Studios. When the Design Director skill creates materials for Arcus, these standards automatically apply to ensure perfect brand compliance while maintaining professional design elevation.

## Brand Identity

**Arcus Innovation Studios** delivers systematic innovation evaluation for organizations addressing the 90% failure rate through data-driven assessment methodology.

**Visual Identity:**
- **Precision** - Systematic, rigorous methodology
- **Credibility** - Trusted advisors at C-level and board level
- **Strategic Insight** - Board-ready decisions and evaluations
- **Professional Excellence** - Executive-grade quality standards

**Primary Colors:**
- **Navy** (#0A2540) - Headers, authority, primary brand
- **Teal** (#00BFB3) - Accents, strategic emphasis, innovation
- **Slate** (#425466) - Secondary text, supporting content
- **Charcoal** (#1A1F36) - Body text, paragraphs

## When This Module Activates

The Arcus brand module automatically activates when:

1. **User mentions "Arcus" or "Arcus Innovation Studios"**
2. **Creating materials for Arcus organization**
3. **User requests Arcus brand standards**
4. **Document is for Arcus clients, partners, or internal use**

## How It Works with Design Director

**Standard Design Director 6-Phase Protocol + Arcus Brand Elements:**

1. **Phase 1 (Functional Foundation):** Standard + identify audience
2. **Phase 2 (Design Interrogation):** Standard + brand requirements
3. **Phase 3 (Technique Selection):** Arcus colors/typography replace exemplars
4. **Phase 4 (Systematic Application):** Apply Arcus elements throughout
5. **Phase 5 (Quality Validation):** Standard + Arcus compliance checks
6. **Phase 6 (Delivery):** Standard + note Arcus brand application

**Key Principle:** Design Director provides the elevation methodology (the "how"), while Arcus brand standards provide the visual identity (the "what").

## Key Brand Elements

### Color Palette

**Primary:**
- Navy (#0A2540)
- Teal (#00BFB3)
- Slate (#425466)

**Neutrals:**
- Charcoal (#1A1F36)
- White (#FFFFFF)
- Gray scale (50, 100, 200, 300, 600, 700, 900)

**Accents (use sparingly):**
- Green (#10B981) - Success
- Amber (#F59E0B) - Warnings
- Red (#EF4444) - Errors
- Blue Light (#0096FF) - Data viz

### Typography

**Hierarchy:**
- H1: 32-36px, Navy, 700 weight
- H2: 24-28px, Navy, 600-700 weight
- H3: 18-20px, Slate/Navy, 600 weight
- Body: 14px, Charcoal, 400 weight, 1.7 line height
- Caption: 12px, Gray 600, 400 weight

**Font Stack:**
- Primary: System UI fonts for cross-platform consistency
- Monospace: SF Mono, Monaco, Cascadia Code, Roboto Mono, Consolas

### Logo

**Minimum Sizes:**
- Print: 0.75 inches (19mm) width
- Digital: 120px width
- Presentations: 1.5 inches (38mm) recommended

**Clear Space:** Height of "A" on all sides

**Variations:**
- Navy on White (standard)
- White on Navy (dark backgrounds)
- Teal on White (digital-only)

### Spacing

**8-Point Grid:** 4, 8, 16, 24, 32, 48, 64, 96px

**Common Applications:**
- Paragraphs: 16px
- Section margins: 48-64px
- Card padding: 24-32px
- Page margins: 48-96px
